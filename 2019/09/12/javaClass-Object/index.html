    类与对象

    java语言的特征：面向对象
    类：java最小执行单元class 是对象的一个模板
    对象：基于类创建的一个实体，每个对象仅仅参照一个类

    创建类

    1.属性 存在在类里面
    2.行为 方法 也是写在类里

    代码示例 学生类

    package com.javasm.odj;

    public class Student {
    /*
    * 构造方法
    * 作用：在创建对象的同时，给对象赋值
    * 特点：
    * 构造方法名与类名相同
    * 构造方法没有返回值，也没有返回值类型
    * 系统默认有一个无参构造(缺省构造器)
    * 快捷键：Alt+Inside
    * */
    String name;//属性
    String number;
    char gender;
    public Student() { //无参构造方法
    }
    public Student(String name, String number, char gender) {//有参构造方法
    this.name = name;
    this.number = number;
    this.gender = gender;
    }
    public void eat() {
    System.out.println(name + “在吃饭”);
    }//类的方法 主要描述一些行为
    public void study() {
    System.out.println(name + “在学习”);
    }
    public void show() {
    System.out.println(“name:” + name + “,number:” + number + “,gender:” + gender);
    }
    }

    创建对象

    语法 类名 对象名 = new 类名();

    代码示例 李四与张三对象

    package com.javasm.odj;
    public class StudentText {
    public static void main(String[] args) {
    Student stu = new Student();//选用无参构造创建对象
    stu.name = “李四”;
    stu.eat();
    Student stu1 = new Student(“张三”,”12345678910”,’男’);
    stu1.eat();
    stu1.show();
    }
    }

    运行结果

    李四在吃饭
    张三在吃饭
    name:张三,number:12345678910,gender:男
