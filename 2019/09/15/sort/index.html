    排序

    冒泡排序

    依次比较两个相邻的元素，如果他们的顺序（如从大到小、首字母从A到Z）错误就把他们交换过来。走访元素的工作是重复地进行直到没有相邻元素需要交换，也就是说该元素列已经排序完成。

    代码示例

    package com.javasm.array;

    import java.util.Arrays;

    public class MaoPao {
    public static void main(String[] args) {
    //定义数组
    int[] arr = {10,2,15,7};
    //for循环 控制排序的轮数的
    for(int i=0;i<arr.length-1;i++) {
    //控制元素 相邻的元素相比
    for(int j=0;j<arr.length-1-i;j++) {
    //判断相邻的两个元素
    if(arr[j]>arr[j+1]) {//arr[j]位置交换
    int temp = arr[j];
    arr[j] = arr[j+1];
    arr[j+1] = temp;
    }
    }
    }
    System.out.println(arr);//不认识 地址值 引用数据类型
    //Arrays
    System.out.println(Arrays.toString(arr));//将数组变量转换称字符串常量
    }
    }

    选择排序

    第一次从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置，然后再从剩余的未排序元素中寻找到最小（大）元素，然后放到已排序的序列的末尾。以此类推，直到全部待排序的数据元素的个数为零。选择排序是不稳定的排序方法。

    代码示例

    package com.javasm.array;

    import java.util.Arrays;

    public class XuanZe {
    public static void main(String[] args) {
    //选择排序
    int[] arr = {10,2,15,7};
    for(int i=0;i<arr.length-1;i++) {
    int min = arr[i];
    int minIndex = i;
    for(int j=i+1;j<arr.length;j++) {
    if(min>arr[j]) {
    min = arr[j];
    minIndex = j;
    }
    }
    //交换位置
    int temp = arr[i];
    arr[i] = arr[minIndex];
    arr[minIndex] = temp;
    }
    System.out.println(Arrays.toString(arr));
    }
    }
