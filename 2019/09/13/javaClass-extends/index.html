    继承extends

    作用：提高代码的重用性
    发生在类与类之间，类与类属于继承关系。一个类（子类 派生类）只能继承另外一个类（父类 基类）
    子类继承父类内容
    语法：
    public class 子类 extends 父类{
    //子类可以继承父类的成员（private修饰的成员以及父类的构造无法继承）
    }
    Object 超类 所有的类都会继承超级父类

    1.1方法重写 override

    前提：有继承关系
    发生在子类里面，子类重写父类的功能
    方法名 形参 返回值必须与父类保持一致。访问权限修饰符可以>=父类修饰符
    protected：受保护的 可以修饰成员。包内友好 子类友好
    public–>protected–>默认–>pritate
